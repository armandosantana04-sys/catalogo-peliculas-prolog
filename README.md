# Catálogo de Películas - Prolog + PHP

Proyecto escolar que integra Prolog y PHP para mostrar un catálogo de películas dinámico.

## Tecnologías utilizadas

- PHP
- SWI-Prolog
- HTML
- CSS

## Funciones

- Catálogo de películas
- Filtro por categoría
- Consulta dinámica desde Prolog
- Vista detalle de películas
- Comunicación PHP ↔ Prolog
